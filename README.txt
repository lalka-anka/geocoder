Приложение «Геокодер»
Версия beta 2.0
Автор: Ильина Анна (anna.ilina. 21.2000@gmail.com) 

Описание
Вход: адрес в свободной форме
Выход: координаты и полный адрес

Программа не до конца реализована, т.к. производит скачивание файлов по подготовленным  тренировочным ссылкам, а не из открытых архивов.

Требования
Python версии не ниже 3.6
Состав:
• Запуск программы, реализация скачивания данных: geocoder.py
• Реализация создания базы данных: get_data_base.py
• Обработчик информации: process_info.py
• Поиск ссылок: get_refs.py
• Вывод данных в формате json: return_in_json.py
• Тесты: test.py


Справка по запуску:
python3 geocoder.py --help

Пример запуска:

python3 geocoder.py -st Уральская -bld 137 -doc doc_to_save_result
python3 geocoder.py -st Уральская -bld 137
python3 geocoder.py
